Come utilizzare lo script in python:

1. Eseguire lo script.
2. Scrivere da tastiera il path della cartella dove risiedono le acquisizioni (Le acquisizioni devono essere in formato csv).
3. Lo script creerà in quel percorso una cartella denominata "Conversione". Essa conterrà i file csv nel formato richiesto.

Nel caso si voglia rieseguirlo sullo stesso path non occorrerà fare nulla: I file dentro "Conversione" verranno cancellati e sostituiti
con i nuovi appena creati.
